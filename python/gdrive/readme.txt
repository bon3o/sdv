Бизнес-проблема: Регулярные проблемы с пробросом OCT конверсий в Adwords

Бизнес-задача: Реализовать мониторинг конверсий в Adwords

 

Базовый вариант:

Скрипт на Python3 (и должен будет работать под Linux), который выдает список csv файлов с датой их последнего изменения, размером (в байтах) и количества строк в каждом из них в свободном формате

 

Продвинутый вариант:

2 скрипта:

1 скрипт - выводит список файлов в json формате для автообнаружения Zabbix (по аналогии с тем, как ты делал в прошлой задаче)

2 скрипт - на вход получает название файла и метрику (размер/дата обновления/количество строк) и отдает значение

 

Файлы размещены в папке OCT uploads на Google Disk, расшаренном для аккаунта
